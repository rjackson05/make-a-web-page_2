Adding Sytle with CSS
CSS is code written to control the "sytle" of HTML elements This can be done by using elements and class
There are 3 methods to write CSS
Method 1: Write CSS into the HTML by adding: head element and style elements. This is used for small projects
Method 2: Link HTML to a seperate CSS file. This adds additional steps but works well in larger projects.
Method 3: Write style inline with your HTML by coding each individual HTML element. This approach can cause a lot of repeated code.
